# Generic Repository System (Java)
## Project Overview

**This project demonstrates the use of Java Generics, Interfaces, and Collections by implementing a simple Repository pattern for managing different entities.**

*The main goal is to show how to create a type-safe, reusable data manager using generics.*

## Features

- **Generic Repository<T> for storing entities**

- **Entity interface with unique ID**

- **Type-safe data management**

- **Usage of ArrayList and HashMap**

- **Clean object-oriented design**

- **Separation into packages (model, repository)**
